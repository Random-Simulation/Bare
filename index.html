Safety

Disables bash, write, and edit tools entirely. Only read, websearch, and finish_task are available.

When enabled, Bare will only be able to read/write files and run commands within the selected working directory.

⚠ Warning: Disabling this setting allows Bare to work anywhere on your computer.

Bare will ask before running bash, write, or edit. Click "Allow All Tools" on any prompt to disable this.

⚠ Warning: Disabling this allows Bare to run all tools without asking. This includes running system commands and writing, editing and deleting files anywhere on your computer.
